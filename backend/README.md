# Simple Flask Web App

This is a simple Flask web application that displays a welcome message.

## Setup

1. Create a virtual environment and activate it:

    python3 -m venv venv
    source venv/bin/activate

2. Install Flask:

    pip install Flask

3. Run the server:

    python server.py

4. Open your browser and visit `http://127.0.0.1:5000/` to see the welcome message.
